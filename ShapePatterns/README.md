# AIT-Shapes & Patterns

Shapes and patterns assignment.

## Installation
1. Clone the repository:
   ```
   git clone https://github.com/EKasuti/AIT-Computer_Graphics.git
   cd ShapePatterns
   ```
2. Build with gradle:
   ```
   gradle build
   ```
3. Run it locally with a simple server:
   ```
   npx http-server
   // or
   http-server
   ```
